# ip-region-data

ip2region data, from [https://github.com/lionsoul2014/ip2region](https://github.com/lionsoul2014/ip2region).

## Installation

* org.lionsoul:ip2region:2.6.3 -> io.github.ujcms:ip-region-data:2.7.0
* org.lionsoul:ip2region:2.7.0 -> io.github.ujcms:ip-region-data:2.11.2

Add the following dependency to your `pom.xml`:

```xml
<dependency>
    <groupId>io.github.ujcms</groupId>
    <artifactId>ip-region-data</artifactId>
    <version>2.11.2</version>
</dependency>
```
